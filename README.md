# Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecutar el siguiente comando.
```
yarn install
```
3. Tener Nest CLI instalado. Si no lo tiene, puede instalarlo as√≠:
```
npm i -g @nestjs/cli
```

4. Levantar la base de datos.
```
docker-compose up -d
```
